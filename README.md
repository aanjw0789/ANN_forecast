## About This Repository

This repository contains the Python codes developed for the study:

**Neural Network-Based Prediction of Organic Carbon Dynamics in a Monsoon-Dominated Coastal System**  
by A'an Johan Wahyudi and Febty Febriani  
Accepted in *Estuarine, Coastal and Shelf Science*.

The workflow was designed to investigate and predict the variability of organic carbon dynamics in a monsoon-influenced coastal environment using neural network approaches and environmental predictor variables.

The repository includes:

- Data preprocessing and quality control
- Feature engineering and normalization
- Neural network model development and training
- Model evaluation and validation
- Spatio-temporal prediction workflows
- Visualization of prediction outputs and variability patterns

The codes were developed using Python and several scientific computing libraries including NumPy, Pandas, Xarray, Scikit-learn, TensorFlow/Keras, and Matplotlib.

## Scientific Objective

The main objective of this study is to improve understanding and prediction of coastal organic carbon dynamics under monsoonal forcing using data-driven approaches.

The model integrates environmental and biogeochemical variables to capture temporal and spatial variability associated with seasonal processes.

## Main Variables

Depending on the workflow, the model may use variables such as:

- Chlorophyll-a (Chl-a)
- Phytoplankton carbon (PhyC)
- Net primary productivity (NPP)
- Sea surface temperature (SST)
- Salinity
- Nutrients
- Dissolved oxygen
- Particulate organic carbon (POC)

## Model Framework

The repository implements neural network-based prediction approaches including:

- Feed-forward Artificial Neural Networks (ANN)
- Time-series forecasting workflows
- Multivariate environmental prediction

## Reproducibility

The repository is intended to support scientific transparency and reproducibility. Users are encouraged to cite the associated publication when using the code.
